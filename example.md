# Демонстрация разных синтаксисов для кастомных слайдов

## Обычный слайд
Стандартный Markdown контент

## Вариант 1: Тройные проценты
%%%
%layout: two_columns
%color: accent
## Слайд с двумя колонками
**Левая колонка:**
- Пункт 1
- Пункт 2

**Правая колонка:**
- Дополнительная информация
- Еще один пункт
%%%
